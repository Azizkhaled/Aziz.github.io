Transforming Text: NLP and Transformers Exploration

Welcome to the "Transforming Text: NLP and Transformers Exploration" project. This project is collection of notebooks I built that delves into the realm of Natural Language Processing (NLP) and its intersection with Transformers, offering an in-depth exploration of various techniques and applications.

The repository begins with a focus on NLP pre-processing, detailing fundamental techniques such as stopwords removal, tokenization, stemming, lemmatization, and Unicode normalization. Building on this foundation, the project delves into attention mechanisms in NLP, explaining concepts like encoder-decoder attention, self-attention, bidirectional attention, and multi-head attention.

Moving into practical applications, the repository covers sentiment analysis using flair and Transformers, named entity recognition with Spacy, open-domain question answering with Elasticsearch, and even extends to projects like analyzing sentiment and organizational entities from Reddit posts. Moreover, it explores similarity search techniques using Transformers and PyTorch. By combining theoretical explanations with hands-on implementation, "Transforming Text" serves as a valuable resource for those looking to understand the nuances of NLP and harness the potential of Transformers in real-world scenarios.
